# Developer Portfolio

A premium one-page developer portfolio and resume built with React, TypeScript, Tailwind CSS, and Framer Motion.

## Features

- Single-page layout with Hero, About, Skills, Tech Stack, Projects, Experience, Marketing, Terminal, and Contact sections
- Dark/light mode with localStorage persistence
- Scroll progress bar and active navigation highlighting
- Framer Motion animations (fade-in, hover lift, counters, typewriter)
- ATS-friendly PDF resume generation via **Download Resume**
- Lazy-loaded sections for performance
- Fully responsive and accessible

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Customize

Update your personal info in:

- `src/config/site.ts` — email, social links, stats
- `src/data/portfolio.ts` — skills, projects, experience content

## Build

```bash
npm run build
npm run preview
```

## Tech Stack

React · TypeScript · Tailwind CSS · Framer Motion · Lucide Icons · React Icons · html2pdf.js · React Hook Form
